### Usage


```sh
npm i -g graphql-ui

 or

npm i --save-dev graphql-ui
```

Run the output reports
```sh
$ graphql-ui https://phoenix-react-apollo-demo.herokuapp.com/graphql

$ ls graphql-ui-reports/
index.html
```
